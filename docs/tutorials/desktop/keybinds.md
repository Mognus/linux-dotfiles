# Desktop Keybinds

This file explains the most important desktop interactions at a high level.

## Focus

- Move focus between windows with directional keybinds
- Keep one hand on the modifier and one hand on the home row

## Windows

- Spawn a terminal quickly
- Close windows without touching the mouse
- Toggle floating mode only when needed
- Use fullscreen intentionally for focused work, not as the default layout

## Workspaces

- Jump directly to numbered workspaces
- Move the current window to another workspace without interrupting flow
- Keep workspace meaning stable, for example:
  - one for coding
  - one for terminal-heavy tasks
  - one for browser/docs
  - one for media or background apps

## Locking and visuals

- Lock the session before leaving the desk
- Keep wallpaper, lock screen, and bar styling visually consistent

## Recommendation

Do not try to memorize every bind at once. Learn the ones for:

- opening a terminal
- changing focus
- closing a window
- switching workspaces
- moving windows between workspaces

Once those feel natural, the rest becomes much easier to absorb.
