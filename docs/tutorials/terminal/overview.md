# Terminal Overview

Alacritty is the terminal. Tmux keeps sessions alive.

## Does

- Fast terminal startup
- Nerd Font rendering
- Transparent desktop-integrated window
- Tmux sessions with vi-style copy mode

## Keybinds

| Keybind | Does |
|---|---|
| `SUPER+Return` | Open terminal |

Shell aliases:

| Alias | Does |
|---|---|
| `tn` | New tmux session |
| `ta` | Attach tmux session |
| `tk` | Kill tmux session |

## Workflow

Use Alacritty for the window, tmux for persistent work, and named sessions for separate projects or services.
