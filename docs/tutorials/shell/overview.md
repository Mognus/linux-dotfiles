# Shell Overview

Zsh is the daily shell.

## Does

- Completion with menu selection
- Shared, deduplicated history
- Autosuggestions and syntax highlighting
- FZF default keybinds
- Zoxide directory jumping
- Compact transient prompt after commands

## Keybinds

| Keybind | Does |
|---|---|
| `Ctrl+R` | Search command history |
| `Ctrl+T` | Search files |
| `Alt-C` | Jump to directory |
| `Up/Down` | Search matching history |
| `Ctrl+Left/Right` | Move by word |
| `Ctrl+W` | Delete back to word or slash |

## Workflow

Type a hint, use history/search instead of retyping, jump directories with `z`, and keep repeated commands behind aliases.
