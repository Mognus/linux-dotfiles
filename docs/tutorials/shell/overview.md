# Shell Overview

The shell configuration is focused on everyday productivity and low-friction command-line work.

## Main file

- `.zshrc`

## What this topic covers

- Aliases and helper functions
- Prompt and shell startup behavior
- Environment defaults
- Quality-of-life improvements for navigation and command history

## Workflow idea

The shell should reduce repetition. Reused commands should become short aliases or small helpers, while the base setup stays readable enough to maintain without guessing what happens during startup.
