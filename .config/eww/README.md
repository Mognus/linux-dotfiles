# EWW

EWW is the small custom widget layer.

## Does

- Shows the `tux-angel` desktop overlay
- Uses EWW state for visibility
- Uses scripts for toggle/hide behavior

## Keybinds

- `Ctrl+SUPER+Shift+T`: toggle `tux-angel`
- Hover: hide `tux-angel`

## Workflow

Keep EWW for small desktop accents or focused widgets. Put larger desktop behavior in Hyprland or Waybar.
